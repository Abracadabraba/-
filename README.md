# 大音 (DaYin) — 双语语音互译对讲 App v3.0

出国旅行用的双语对讲翻译 App（Java 原生 Android）。
顶部选两种语言，两个"按住说话"按钮各代表一方，说完自动识别、翻译、朗读给对方听。
识别/翻译/朗读的方案可以三选一，设置里切换，其中一种**完全不需要信用卡**。

## 三种方案怎么选

| 方案 | 要不要卡 | 要不要注册 | 效果 |
|---|---|---|---|
| **本地识别+朗读 + 百度翻译**（推荐） | 不需要 | 百度账号 + 实名认证（身份证，不是银行卡） | 用手机自带的语音识别/朗读，翻译走百度，够用，冷门语言的朗读音色可能略机械 |
| Google Cloud | 需要 | Google 账号 + 绑卡开通结算 | 识别准、朗读音色好，200+语种 |
| Microsoft Azure | 需要 | Azure 账号 + 绑卡开通订阅 | 同上，作为 Google 的备胎 |

**没有信用卡就选第一种**，设置里选"本地识别+朗读 + 百度翻译"，只需要百度账号实名认证，
每月100万字符翻译额度完全免费，识别和朗读用手机自带的，一分钱不用花。

## 使用前准备（本地方案）

1. 打开 [百度翻译开放平台](https://fanyi-api.baidu.com/)，用百度账号登录。
2. 申请【通用文本翻译API】，开发者类型选"个人开发者"。
3. 按提示完成个人实名认证（填身份证信息，不需要银行卡）——认证后解锁每月100万字符免费额度。
4. 在管理控制台能看到 APPID 和密钥，填到 App 设置页对应的两个输入框。
5. 如果朗读没声音：去手机 设置 → 语言与输入法（或"更多设置"）→ 文字转语音输出，
   检查有没有装目标语言的语音包，没有的话连 Wi-Fi 下载一下（出发前在国内先下载好）。
6. 如果识别没反应：确认手机上有语音识别服务（大部分安卓机自带，国行阉割版有些没有，
   出发前可以先在安静环境测一次）。

## 使用前准备（如果以后办了信用卡，想换 Google/Azure）

### Google Cloud
1. 打开 [Google Cloud Console](https://console.cloud.google.com/)，新建一个项目。
2. 启用：Cloud Speech-to-Text API、Cloud Translation API、Cloud Text-to-Speech API。
3. 创建一个 API Key。
4. 设置页选择"Google Cloud"，填入这个 Key。

免费额度（每月重置）：语音识别60分钟 + 翻译50万字符 + 朗读400万字符。

### Microsoft Azure
1. 打开 [Azure Portal](https://portal.azure.com/)，创建一个 **语音服务(Speech)** 资源（选 F0 免费版），记下 Key 和 Region。
2. 再创建一个 **翻译器(Translator)** 资源（选 F0 免费版），记下 Key 和 Region。
3. 设置页选择"Microsoft Azure"，分别填入这两组 Key/Region。

免费额度（每月重置）：语音识别5小时 + 翻译200万字符 + 朗读50万字符。

## 使用方法

1. 顶部两个下拉框选语言A、语言B（比如 中文 / 越南语）。
2. 谁说话就按住对应的按钮（"按住说 中文（普通话）" / "按住说 Tiếng Việt"），说完松开。
3. App 自动识别文字、翻译、追加显示到聊天记录，并朗读给对方听。

## 用 GitHub Actions 打包

项目里已经配好 `.github/workflows/build.yml`：

1. 在 GitHub 建一个新仓库（私有的就行，里面不含任何密钥）。
2. push 代码：
   ```bash
   cd DaYin
   git init
   git add .
   git commit -m "大音 v3.0"
   git branch -M main
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   git push -u origin main
   ```
3. push 完 GitHub 会自动触发 Actions（仓库页面 "Actions" 标签能看到构建进度）。
4. 构建成功后，进这次运行记录，最下面 "Artifacts" 里有 `dayin-debug-apk`，
   下载解压得到 `app-debug.apk`，传到手机装上（需要在手机设置里允许"安装未知来源应用"）。

这是 Debug 包，自己装测试没问题；给别人长期用的话可以再让我加正式签名的 Release 构建。

## 用 Android Studio 打开（备选）

用 Android Studio「Open」这个文件夹，首次同步 Gradle 时按提示允许联网，
连接手机（开启 USB 调试）后点运行。

## 已知注意事项

- 本地方案的识别、朗读效果依赖你手机本身的系统语音能力，不同手机品牌/系统版本差异较大，
  出发前务必在国内先把所有要用到的语言测一遍（识别 + 朗读都测），确认没问题再出门。
- 百度翻译的语种代码里，越南语(vie)/日语(jp)/韩语(kor)/法语(fra)/阿拉伯语(ara)/西班牙语(spa)
  这几个用的是百度自己的缩写，不是标准 ISO 代码，代码里已经按官方文档填好；
  印尼语/马来语/印地语/土耳其语这几个我不是100%确定百度的代码，如果测试报错，
  去百度翻译开放平台的语种列表页核对一下，改 `Language.java` 里对应那一行即可。
- Google/Azure 的 Key 目前是明文存在手机本地，自己用没问题；给别人用的正式版建议换成后端中转。
- 出国当地如果网络连不上百度/Google/Azure 的服务器，记得开好当地能用的网络。
